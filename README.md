# Default Risk Model

This project builds an interpretable logistic regression model to estimate probability of default for borrowers.

## Overview
The model prioritizes recall for default cases while maintaining strong precision for non-default borrowers.

## Methods
- Logistic Regression
- Train/test split
- ROC AUC, confusion matrix, and probability analysis

## Use Cases
- Credit underwriting
- Portfolio risk monitoring
- Risk-based pricing

## How to Run
1. Install dependencies
2. Run the script in src/default_risk_model.py
